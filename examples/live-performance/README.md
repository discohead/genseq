# Live Performance Example

Example project demonstrating MIDI controller mappings and scene management for live performance.

## Features

- Multiple scenes (intro, main, breakdown, outro)
- MIDI controller mappings for real-time pattern control
- Scene switching via MIDI triggers

## Setup

1. Connect MIDI controller
2. Update device names in `mappings/` directory
3. Configure scenes in `scenes/` directory
4. Map controller knobs/faders to pattern parameters
